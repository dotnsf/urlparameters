# URL Parameters

Decoder for URL parameters into JSON.


## How to use

- Run application:

  - `$ node app`

- Access to `GET /` with URL parameters:

  - Ex. `http://localhost:8080/?a=1&b=2&c=3&d=4`

- You will see JSON-decoded URL parameters:

  - Ex. `{ "a": "1", "b": "2", "c": "3", "d": "4" }`
